Subversion

The development version can be received via SVN with the following command:

svn checkout svn://svn.berlios.de/supertux/trunk/supertux/


Flexlay Editor Download

There exist two level editors for SuperTux, the built-in one that comes with SuperTux and is available via the in-game menu (available in 0.1.1 and later) and the external one, which is known as Flexlay. Flexlay is a seperate project that aims to provide level editing capabilities for multiple games.
