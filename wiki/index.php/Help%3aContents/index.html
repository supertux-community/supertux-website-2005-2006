From SuperTux

SuperTux is an open source platform game in the style of the Super Mario Bros. (http://en.wikipedia.org/wiki/Super_Mario_Bros.) games created by Nintendo. The following articles contain guidance and information about participating in the SuperTux developer community and using this Wiki.
